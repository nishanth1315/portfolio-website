# Professional Personal Portfolio

A clean, responsive, and static personal portfolio website built with HTML5 and CSS3.

## Features
- **Responsive Design**: Mobile-first approach using Flexbox and Grid.
- **Static**: No JavaScript frameworks or external libraries.
- **PWA Ready**: Includes `manifest.json` and offline fallback page.
- **Pages**: Home, About, Skills, Projects, Contact.

## Setup
1. Unzip the project.
2. Open `index.html` in your web browser.
3. No server or build process required.

## Customization
- Edit `style.css` to change the `--primary-color` or other variables.
- Replace placeholder text in HTML files with your own content.
- Add your images to `assets/images/` and update `src` attributes.
